## 🎯 About Me

저는 **회계사의 본질적 역할 = “신뢰 가능한 숫자를 통해 의사결정을 돕는 것”** 이라고 생각합니다.

이를 위해 다음 3가지 방향으로 역량을 확장하고 있습니다.

1. **재무·회계 전문성 강화**  
   - K-IFRS 기반 재무제표 구조 이해  
   - 감사 관점의 데이터 신뢰성, 내부통제, 추적성(Audit Trail)

2. **ESG · 기후 공시 대응 역량**  
   - ISSB(IFRS S2), TCFD 기반 기후 리스크 → 재무 영향 연결  
   - 전환·물리 시나리오를 **정량 재무 모델**로 구현

3. **데이터 · AI 기반 회계 자동화**  
   - Python 기반 재무·ESG 데이터 처리  
   - 정성 정보의 구조화 및 자동 리포팅

> 목표: **회계사의 전문성을 디지털화하여,  
> 기업 의사결정의 ‘신뢰도’와 ‘속도’를 동시에 높이는 회계·데이터 전문가**
